# 전월세 계산기 기획서

> **Phase**: `Phase 11`
> **한 줄 설명**: 전세금과 월세 조건을 실시간으로 상호 전환하고, 두 옵션의 실질 비용을 비교해 최선의 선택을 도와주는 계산기

---

## 개요

이사나 재계약을 앞둔 세입자·집주인이 전세와 월세 중 어떤 조건이 자신에게 유리한지 즉시 판단할 수 있는 계산기.

- 전세금 ↔ 월세의 **양방향 자동 전환**이 핵심이다.
- 단순 환산에 그치지 않고 **기회비용 기반 실질 비용 비교**를 제공해 의사결정을 돕는다.
- 법정 전월세 전환율을 기본값으로 제공하되, 사용자가 실제 시세에 맞게 수정할 수 있다.
- 전세금·보증금·월세 세 값을 직접 입력하면 **실제 전환율을 자동 역산**해 법정 기준과 비교한다.

> **법정 전월세 전환율 공식** (주택임대차보호법 제7조의2)
> `전환율 상한 = 한국은행 기준금리 + 3.5%` (현재 상한 7.0%)

---

## 핵심 계산 공식

```
월세  = (전세금 - 보증금) × 전환율(연%) ÷ 12
전세금 = 보증금 + (월세 × 12 ÷ 전환율)
전환율 역산 = 월세 × 12 ÷ (전세금 - 보증금) × 100

전세 기회비용(월) = 전세금 × 예금금리(연%) ÷ 12
월세 실질비용(월) = 월세 + 보증금 × 예금금리(연%) ÷ 12
```

## 화면 구조: 고정 상단 + 스와이프 하단

화면을 **고정 영역**(전세금·보증금)과 **스와이프 영역**(페이지 1·2)으로 나눈다.
전세금·보증금이 스와이프해도 그대로 유지되어, 두 페이지가 **같은 조건의 다른 관점**임을 시각적으로 전달한다.

### 페이지 1: 전월세 전환 (주 기능)

전세금 또는 월세 중 하나를 입력하면 나머지가 자동 계산된다.
전환율은 사용자가 직접 설정하는 **파라미터**.

```
──── 고정 영역 ────────────────────────────
┌─── 전세금 ──────────────────────────┐
│       3억 0000 만원                  │  입력 모드
└──────────────────────────────────────┘

  보증금       5000 만원           ▷

──── 스와이프 영역 (페이지 1) ──────────
  전월세 전환                  ● ○

  전환율       5.00 %              ▷       파라미터
                ⇅
┌─── 월 세 ────────────────────────────┐
│          = 104 만원                   │  결과 모드
│   전세 대비 월 21만원 유리  · ▷        │  비용 비교 인라인 요약
└───────────────────────────────────────┘
```

**전세금 카드 활성 시** (전세금 입력 → 월세 자동 계산):

```
┌─── 전세금 ──────────────────────────┐  입력 모드
│  [커서]  5억 0000 만원               │  테두리 강조, 키패드 연결
└──────────────────────────────────────┘

  보증금       1억 0000 만원       ▷      공유 파라미터 (수정해도 방향 유지)
  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
  전환율       5.00 %              ▷
                ⇅
┌─── 월 세 ────────────────────────────┐  결과 모드
│          = 166 만원                   │  흐린 배경, 포인트 컬러
│   전세 대비 월 21만원 유리  · ▷        │
└───────────────────────────────────────┘
```

**월세 카드 활성 시** (월세 입력 → 전세금 자동 계산):

```
┌─── 전세금 ──────────────────────────┐  결과 모드
│          = 4억 9920 만원              │  흐린 배경, 포인트 컬러
└──────────────────────────────────────┘

  보증금       1억 0000 만원       ▷
  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
  전환율       5.00 %              ▷
                ⇅
┌─── 월 세 ────────────────────────────┐  입력 모드
│  [커서]  150 만원                     │  테두리 강조, 키패드 연결
└───────────────────────────────────────┘
```

### 페이지 2: 전환율 확인 (조건 검증)

전세금·보증금·월세 세 값을 직접 입력하면 전환율을 자동 역산한다.
"집주인이 제시한 조건이 법적으로 적정한가?"를 확인하는 데 쓴다.

```
──── 고정 영역 ────────────────────────────
┌─── 전세금 ──────────────────────────┐
│       3억 0000 만원                  │  입력
└──────────────────────────────────────┘

  보증금       5000 만원           ▷

──── 스와이프 영역 (페이지 2) ──────────
                    전환율 확인  ○ ●

┌─── 월 세 ────────────────────────────┐
│          150 만원                     │  입력 (직접 입력)
└───────────────────────────────────────┘

  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  적용 전환율           = 7.20 %          자동 계산 결과
  법정 상한               7.00 %
  ⚠ 법정 상한을 0.20%p 초과
```

### 두 페이지 비교

| 항목 | 페이지 1: 전월세 전환 | 페이지 2: 전환율 확인 |
|------|----------------------|----------------------|
| **목적** | 전세↔월세 금액 변환 | 제시 조건의 전환율 검증 |
| **전세금** | 입력 또는 결과 (방향에 따라) | 항상 입력 (고정 영역에서 공유) |
| **보증금** | 공유 파라미터 | 공유 파라미터 |
| **전환율** | 파라미터 (사용자가 설정) | **결과** (자동 역산) |
| **월세** | 입력 또는 결과 (방향에 따라) | 항상 입력 (직접 입력) |
| **⇅ 스왑 버튼** | 있음 (양방향 전환) | 없음 (두 카드 모두 입력) |
| **비용 비교** | 결과 카드에 인라인 요약 | 없음 (전환율 검증이 목적) |
| **법정 상한 비교** | 전환율 행에 서브텍스트 | 역산 결과 아래에 명시적 비교 |

---

### 스와이프 전환 시 데이터 흐름

전세금·보증금은 고정 영역에 물리적으로 존재하므로 **탭 간 공유가 설명 없이 자명**하다.

| 전환 방향 | 데이터 처리 |
|----------|------------|
| 페이지 1 → 페이지 2 | 전세금·보증금 그대로 유지, 월세는 페이지 2의 이전 값 유지 (첫 진입 시 비어있음) |
| 페이지 2 → 페이지 1 | 전세금·보증금 그대로 반영, 전환율은 페이지 1의 기존 설정값 유지, 월세 재계산 |
| 페이지 2에서 전세금 수정 | 고정 영역이므로 양쪽 페이지에 즉시 반영 |

---

### 카드 상태: 입력 모드 vs 결과 모드

페이지 1에서 **어느 카드를 탭했는가**로만 계산 방향이 결정된다.
결과 카드를 탭하면 ⇅ 스왑과 동일하게 입력/결과 교체가 일어난다 (스왑 애니메이션 제공).

| 상태 | 카드 스타일 | 텍스트 |
|------|------------|--------|
| 입력 모드 | 테두리 강조 (테마 컬러) | 기본 텍스트 컬러 |
| 결과 모드 | 흐린 테두리, 배경 약간 어둡게 | 포인트 컬러 + `=` 접두 표시 |

### 공유 파라미터 행

```
  보증금    1억 0000 만원    ▷    ← 탭하면 키패드로 수정 (방향 변경 없음)
  전환율        5.00 %      ▷    ← 탭하면 소수점 키패드로 수정 (페이지 1에서만)
```

- 보증금은 고정 영역에, 전환율은 페이지 1 스와이프 영역에 위치
- 탭해서 수정해도 현재 입력/결과 카드 역할이 유지됨
- 파라미터 수정 완료(`완료` 탭) 시 이전 활성 카드로 포커스 복귀
- 전환율 옆에 `법정 상한 7.0%` 서브텍스트로 안내
- 입력한 전환율 > 법정 상한일 경우: ⚠ 경고 아이콘 표시

---

### ⇅ 스왑 버튼 (페이지 1 전용)

- 탭하면 입력 모드 카드와 결과 모드 카드를 **명시적으로 교체**
- 결과 카드를 직접 탭해도 동일하게 스왑 (보증금이 독립 파라미터이므로 안전)
- 스왑 시 카드 위치 교환 애니메이션 제공 → 실수로 탭해도 변화를 즉시 인지
- 페이지 2에서는 ⇅ 버튼 없음 (두 카드 모두 입력 모드)

### 비용 비교

#### 인라인 요약 (페이지 1 결과 카드 내)

키패드가 올라와도 핵심 결론이 항상 보이도록, 결과 카드 안에 한 줄로 포함한다.

```
┌─── 월 세 ────────────────────────────┐  결과 모드
│          = 166 만원                   │
│   전세 대비 월 21만원 유리  · ▷        │  ← 인라인 요약 (항상 보임)
└───────────────────────────────────────┘
```

- `▷` 탭 시 키패드 닫히고 상세 비용 비교 카드 표시

#### 상세 비용 비교 카드 (스크롤 영역)

```
┌─── 실질 비용 비교 ──────────────────────┐
│                                          │
│  전세 기회비용   ████████░░   145만원/월  │  ← 전세금 × 예금금리 ÷ 12
│  월세 실질비용   ██████████   166만원/월  │  ← 월세 + 보증금 × 예금금리 ÷ 12
│                                          │
│  ────────────────────────────────────── │
│  월세가 월 21만원 더 유리합니다            │  ← 차이 강조
│                                          │
│  예금 금리   3.50 %                수정 ▷ │  ← 탭하면 수정
└──────────────────────────────────────────┘
```

- 막대 그래프로 두 비용의 시각적 비율 확인
- 비교 결과: "전세가 유리" / "월세가 유리" / "동일"
- 예금 금리 기본값 3.50%, 수정 시 즉시 재계산

### 숫자 키패드

```
[ 7 ] [ 8 ] [ 9 ] [   ←   ]
[ 4 ] [ 5 ] [ 6 ] [  000  ]    ← 0 세 자리 입력 (천만원 단위)
[ 1 ] [ 2 ] [ 3 ] [   억   ]
[  00  ] [  0  ] [  C  ] [ 완료 ]
```

#### `억` 버튼 동작

```
규칙: [숫자 입력] → [억] → 현재 숫자 × 10,000 확정
                            이후 숫자는 만원 단위로 추가

예시:
  5  →  억                    = 50,000만원  (5억)
  5  →  억  →  3  →  000     = 53,000만원  (5억 3,000만원)
  1  →  억  →  5  →  000     = 15,000만원  (1억 5,000만원)
  3  →  000                   = 3,000만원   (3,000만원)
```

- `억` 버튼: 현재 입력 버퍼 × 10,000 확정 후, 추가 입력은 만원 단위로 누적
- 버퍼가 비어있을 때 `억`: 무시
- 이미 억 확정된 상태에서 `억` 다시: 무시
- `000` 버튼: 0 세 자리 입력 (1000만원, 3000만원 빠른 입력용)
- `C` 버튼: 현재 입력 필드 전체 클리어

#### 금액 표시 포맷 규칙

```
입력 중 (버퍼 표시):   53000 만원
입력 완료 (포맷 표시): 5억 3,000만원
1억 미만:              3,000만원
1억 단위 정확:         3억
```

금액 필드는 **만원 단위 단일 정수**로 내부 저장한다.

## 기능 요구사항

### 입력

- 전세금 카드 / 월세 카드를 탭하면 해당 카드가 **입력 모드**로 전환, 키패드가 올라옴
- 페이지 1: 결과 카드를 탭하면 ⇅ 스왑과 동일하게 입력/결과 교체 (스왑 애니메이션)
- 페이지 2: 전세금·월세 두 카드 모두 입력 가능, 탭한 카드에 키패드 포커스
- 보증금 행을 탭하면 해당 값만 수정, 카드 방향은 유지 → `완료` 시 이전 활성 카드로 포커스 복귀
- 키패드 `완료` 버튼 또는 배경 탭으로 키패드 닫힘
- 초기 진입 시: 기본값이 채워진 상태로 표시, 키패드 자동 올라오지 않음

### 계산 방향 (페이지 1)

| 활성 카드 | 계산 방향 | 파라미터 |
|----------|----------|---------|
| 전세금 카드 활성 | 전세금 → 월세 계산 | 보증금, 전환율 고정 |
| 월세 카드 활성 | 월세 → 전세금 역산 | 보증금, 전환율 고정 |
| 보증금 수정 | 현재 활성 카드 기준 재계산 | 방향 유지 |
| 전환율 수정 | 현재 활성 카드 기준 재계산 | 방향 유지 |

### 전환율 자동 역산 (페이지 2)

- 전세금·보증금·월세 세 값이 모두 0이 아닌 값일 때 전환율을 역산해 결과 영역에 표시
- 법정 상한(7.0%)과 비교해 초과 여부를 명시적으로 안내

### 표시

- 금액: 억/만원 혼합 표기 (`5억 3,000만원`, `3,000만원`, `5억`)
- 입력 중: `53000 만원` (포맷 없는 숫자 + 단위)
- 계산 결과: 1만원 미만 반올림
- 결과 모드 카드 텍스트 앞에 `=` 접두 표시

### 비용 비교

- 전세 기회비용 = `전세금 × 예금금리 ÷ 12`
- 월세 실질비용 = `월세 + 보증금 × 예금금리 ÷ 12`
- 예금 금리 수정 시 즉시 재계산
- 결과 카드에 한 줄 인라인 요약 항상 표시, 상세 카드는 스크롤 영역

## 예외 / 엣지 케이스

| 상황 | 기대 동작 |
|------|-----------|
| 보증금 ≥ 전세금 | 월세 = 0 표시, "보증금이 전세금 이상입니다" 안내 |
| 전세금 = 0 | 결과 0 표시, 비용 비교 미표시 |
| 전환율 = 0 (페이지 1) | 계산 불가, "전환율을 입력해주세요" 안내 |
| 예금 금리 = 0 | 기회비용 = 0, 월세 실질비용 = 월세만 표시 |
| 계산 결과 소수 | 1만원 미만 반올림 |
| 월세 역산 결과 음수 | "조건을 다시 확인해주세요" 안내 |
| 전환율 > 법정 상한 | ⚠ 아이콘 + "법정 상한(7.0%) 초과" 경고 |
| 페이지 2에서 전세금 - 보증금 ≤ 0 | 전환율 = 0 표시, "전세금이 보증금보다 커야 합니다" 안내 |
| 페이지 2에서 월세 미입력 | 역산 결과 미표시, "월세를 입력해주세요" 안내 |
