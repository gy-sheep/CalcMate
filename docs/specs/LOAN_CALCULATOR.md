# 대출 계산기 스펙

> **Phase**: `Phase 7`
> **한 줄 설명**: 대출 조건을 입력하면 월 납입금·총 이자·상환 일정을 즉시 확인할 수 있는 계산기
> **상태**: `화면 구성 검토 중` — 4가지 안 중 최종 선택 후 상세 스펙 작성 예정

---

## 개요

주택 구매, 자동차 할부, 신용 대출 등을 앞둔 사용자가 다음 질문에 즉각적으로 답을 얻는 계산기.

- "이 조건이라면 매달 얼마나 나가지?"
- "월 150만 원을 낼 수 있다면 얼마까지 빌릴 수 있지?"
- "금리가 오르면 총 이자가 얼마나 늘어나지?"

---

## 화면 구성 비교 (4가지 안)

---

### 안 A — 표준 입력형 *(최초 제안)*

#### 기획 의도

입력 → 결과 순서의 전통적인 계산기 흐름. 사용자가 조건을 먼저 설정한 뒤 결과를 확인한다. 구조가 익숙해 학습 비용이 낮고, 모든 입력값을 한 화면에서 파악할 수 있다. 단, "입력하는 도구"라는 느낌이 강해 탐색적 사용(이 조건은 어때?)에는 다소 부자연스럽다.

#### 화면

```
┌─────────────────────────────────┐
│  대출 계산기                     │
├─────────────────────────────────┤
│                                 │
│  대출 원금                      │
│  ━━━━━━━●━━━━━━━━  [ 1억 원 ▾] │
│  1천만              10억        │
│                                 │
│  연 이율                        │
│  ━━●━━━━━━━━━━━━━  [ 3.50 % ▾] │
│  0.1%              20%          │
│                                 │
│  대출 기간                      │
│  ━━━━━━━━━━━━━●━━  [  30 년  ▾] │
│  1년               40년         │
│                                 │
│  상환 방식                      │
│  ◉ 원리금균등  ○ 원금균등  ○ 만기일시 │
│                                 │
├─────────────────────────────────┤
│  월 납입금          ₩1,234,567  │  ← 결과 (하단)
│  총 상환금        ₩148,148,040  │
│  총 이자           ₩48,148,040  │
│  ▓▓▓▓▓▓▓▓▓▓▓▓░░░░  원금 67%    │
├─────────────────────────────────┤
│  상환 일정  ▾ 펼치기            │
│  회차  납입금  원금  이자  잔금  │
│    1   ...    ...   ...   ...   │
└─────────────────────────────────┘
```

---

### 안 B — 결과 우선형

#### 기획 의도

앱 진입 즉시 기본값으로 계산된 결과를 먼저 보여주고, 아래 슬라이더로 조건을 조정하는 구조. "계산기를 사용한다"는 느낌보다 "시뮬레이션한다"는 느낌을 준다. 슬라이더를 움직일 때마다 상단 결과가 실시간으로 바뀌어, 금리나 기간이 납입금에 미치는 영향을 직관적으로 체감할 수 있다.

#### 화면

```
┌─────────────────────────────────┐
│  대출 계산기                     │
├─────────────────────────────────┤
│                                 │
│  ┌───────────────────────────┐  │
│  │  월 납입금                │  │  ← 결과가 먼저
│  │                           │  │
│  │    ₩ 1,234,567            │  │
│  │                           │  │
│  │  총 상환금  ₩148,148,040  │  │
│  │  총 이자     ₩48,148,040  │  │
│  │  ▓▓▓▓▓▓▓▓▓▓░░░  67%/33%  │  │
│  └───────────────────────────┘  │
│                                 │
├─────────────────────────────────┤
│                                 │
│  대출금   ━━━━━●━━━━━  1억     │  ← 조건 조절
│  이율     ━●━━━━━━━━━  3.5%   │
│  기간     ━━━━━━━━━●━  30년   │
│                                 │
│  [원리금균등]  [원금균등]  [만기일시] │
│                                 │
├─────────────────────────────────┤
│  상환 일정 보기  →              │
└─────────────────────────────────┘
```

### 안 C — 타임라인형

#### 기획 의도

대출을 "매달 얼마씩 내는 숫자" 대신 "30년이라는 시간 여정"으로 시각화한다. 연도별 누적 상환액 바 차트(원금/이자 구분)를 중심에 배치해, "10년 후엔 원금을 얼마나 갚은 거지?", "절반을 갚으려면 몇 년이야?"를 직관적으로 파악할 수 있다. 타임라인 바를 탭하면 해당 시점의 잔금과 누적 상환액을 툴팁으로 보여준다.

#### 화면

```
┌─────────────────────────────────┐
│  대출 계산기                     │
├─────────────────────────────────┤
│  월 납입금           ₩1,234,567 │  ← 결과 (컴팩트)
│  총 이자             ₩48,148,040│
├─────────────────────────────────┤
│                                 │
│  상환 여정                      │
│                                 │
│  납│                    ████    │
│  입│              ████████████  │
│  금│        ██████████████████  │
│  누│  ██████████████████████████│
│  적│──┬─────┬──────┬──────┬────│
│    │ 5년   10년    20년   30년  │
│                                 │
│    ■ 원금  ■ 이자              │
│                                 │
│         ↑ 탭 → 해당 시점 잔금   │
│                                 │
├─────────────────────────────────┤
│  대출금   ━━━━━●━━━━━  1억     │
│  이율     ━●━━━━━━━━━  3.5%   │
│  기간     ━━━━━━━━━●━  30년   │
│  상환 방식  [원리금균등 ▾]       │
└─────────────────────────────────┘
```

---

### 안 D — 목표 역산형

#### 기획 의도

"나는 얼마를 빌리고 싶다"가 아니라 "나는 매달 이만큼 낼 수 있다"로 시작하는 구조. 대출이 처음인 사용자나, 이미 월 예산이 정해진 사용자에게 훨씬 자연스러운 출발점이다. 기본 모드가 역산(월납입금 → 최대 대출금)이고, 상단 탭으로 일반 계산(대출금 → 월납입금)으로 전환할 수 있다.

#### 화면

```
┌─────────────────────────────────┐
│  대출 계산기                     │
├─────────────────────────────────┤
│  [최대 대출금 계산]  [납입금 계산] │  ← 모드 탭
├─────────────────────────────────┤
│                                 │
│  매달 낼 수 있는 금액은?         │
│                                 │
│  ┌───────────────────────────┐  │
│  │   ₩ 1,200,000             │  │  ← 직접 입력 (포커스 상태)
│  └───────────────────────────┘  │
│                                 │
│  이율     ━●━━━━━━━━━  3.5%   │
│  기간     ━━━━━━━━━●━  30년   │
│  상환 방식  [원리금균등 ▾]       │
│                                 │
├─────────────────────────────────┤
│                                 │
│  최대 대출 가능금액              │
│                                 │
│       ₩ 267,181,756             │  ← 역산 결과 (크게)
│                                 │
│  총 상환금  ₩432,000,000        │
│  총 이자    ₩164,818,244        │
│                                 │
└─────────────────────────────────┘
```

- 예외 / 엣지 케이스
