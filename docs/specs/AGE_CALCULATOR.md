# 나이 계산기 스펙

> **Phase**: `Phase 5`
> **한 줄 설명**: 생년월일(양력/음력 선택)을 입력하면 만 나이·세는 나이·띠·별자리 등 나와 관련된 시간 정보를 한눈에 보여주는 나이 계산기

---

## 개요

사용자가 생년월일을 드럼롤 피커로 선택하면, 만 나이(법정 나이)를 중심으로
세는 나이·연 나이·다음 생일까지 D-day·살아온 날수·띠·별자리를 실시간으로 표시한다.

**양력/음력 모드**를 지원한다. 음력 모드에서는 입력한 음력 날짜를 내부적으로 양력으로 변환하여
나이 계산 및 D-day를 계산하며, 다음 생일도 음력 기준으로 추적한다.
띠는 음력 설날(정월 초하루) 기준으로 계산하므로 양력 1~2월 출생자의 경우 연도만으로 추정하는
방식보다 정확한 결과를 제공한다.

키패드 없이 날짜 피커만으로 모든 입력이 이루어지며,
결과는 정보 유형별로 그룹화된 카드로 제공하여 빠르게 원하는 정보를 확인할 수 있다.

> **만 나이 우선**: 2023년 6월부터 한국 법령이 만 나이를 공식 기준으로 채택했으므로,
> 만 나이를 가장 크고 먼저 표시한다.

## 기능 요구사항

### 양력/음력 모드 전환

- 피커 위에 **[ 양력 ] / [ 음력 ]** 세그먼트 컨트롤을 배치한다
- 모드 전환 시 피커는 초기값(30년 전)으로 리셋한다
- 모드에 따라 입력 라벨이 "생년월일" / "생년월일 (음력)"으로 변경된다

---

### 생년월일 입력 (드럼롤 피커)

**공통**
- 화면 상단 고정 영역에 **연·월·일** 드럼롤 피커 3개를 가로 배치한다
- 피커를 스크롤하면 날짜가 즉시 변경되고 결과가 실시간으로 업데이트된다
- **연도 범위**: 1900년 ~ 오늘의 연도
- **기본값**: 최초 진입 시 오늘로부터 30년 전 날짜

**양력 모드**
- **월**: 01 ~ 12
- **일**: 해당 연·월의 실제 일수에 맞게 동적 조정 (2월 28/29일)
- 미래 날짜(오늘 이후) 선택 불가 — 오늘 이후 날짜로 이동 시 오늘로 클램프

**음력 모드**
- **월**: 01 ~ 12 (음력 기준)
- **일**: 음력 해당 월의 실제 일수 (대개 29일 또는 30일), 연·월 조합에 따라 동적 조정
- **윤달 체크박스**: 선택한 연·월에 음력 윤달이 존재하는 경우에만 체크박스를 활성화하여 표시
  - 체크 시 해당 윤달(예: 윤 11월)의 날짜로 변환
  - 해당 연·월에 윤달이 없으면 체크박스를 비활성화(dimmed)하여 표시
- 피커 아래에 **변환된 양력 날짜**를 소형 텍스트로 실시간 표시한다 (예: `양력 1995년 11월 23일`)
- 음력 변환에는 1900 ~ 2100 범위의 **하드코딩 룩업 테이블**을 사용한다 (네트워크 미사용)

---

### 나이 결과 카드

세 가지 나이를 하나의 카드에 표시한다. 음력 모드에서도 **변환된 양력 날짜** 기준으로 동일하게 계산한다.

| 항목 | 계산 방식 | 표시 예 |
|------|-----------|---------|
| 만 나이 | 오늘 기준 생일이 지났으면 `올해 − 출생연도`, 아직이면 `올해 − 출생연도 − 1` | `30세` |
| 세는 나이 | `올해 − 출생연도 + 1` | `32세` |
| 연 나이 | `올해 − 출생연도` | `31세` |

- **만 나이**는 카드 최상단에 가장 큰 폰트(28sp 이상)와 포인트 컬러로 강조한다
- 세는 나이·연 나이는 구분선 아래 보조 텍스트 크기(16sp)로 표시한다

> **음력 모드 만 나이 계산 기준**: 음력 생일이 오늘 기준으로 지났는지는
> 음력 생일을 이번 해 양력 날짜로 변환한 값과 오늘을 비교하여 판단한다.

---

### 다음 생일 카드

**양력 모드**
- **오늘이 생일**: "🎂 오늘이 생일이에요!" 메시지를 카드 전체에 표시 (골드 강조)
- **생일 아닌 경우**: "D-{N}" 형식 + 서브텍스트에 다음 생일 "YYYY년 MM월 DD일 (요일)"

**음력 모드**
- 카드 라벨: "다음 음력 생일"
- 다음 음력 생일(음력 기준)을 양력으로 변환하여 D-day 계산
- 서브텍스트: 다음 음력 생일의 양력 날짜 + 음력 날짜를 병기 (예: `양력 11월 23일 (음력 11.01)`)
- **오늘이 음력 생일**: "🎂 오늘이 음력 생일이에요!" 표시

**공통**
- 다음 생일은 오늘 이후의 가장 가까운 생일로 계산한다
- 오늘이 생일인 경우, 내년 생일은 D-day가 아닌 축하 메시지로 대체한다

---

### 살아온 날 카드

- 출생일(양력 기준, 음력 모드는 변환 후 양력)부터 오늘까지 경과한 **총 일수**를 표시한다
- 표시 형식: `N,NNN일`
- 서브텍스트: 약 몇 년 몇 개월인지 보조 표시 (예: `약 31년 3개월`)

---

### 띠 카드

- 표시 형식: `{이모지} {동물}띠` (예: `🐲 용띠`)
- **양력 모드**: 양력 연도 기준으로 계산하되, 출생일이 해당 연도의 음력 설날 이전이면 **전년도 띠**로 표시한다
- **음력 모드**: 입력한 음력 날짜를 양력으로 변환한 뒤, 동일한 설날 기준 로직을 적용한다

> **띠 계산 상세**:
> 음력 설날(정월 초하루)은 매년 양력 1월 21일 ~ 2월 20일 사이에 위치한다.
> 해당 연도의 설날 양력 날짜는 룩업 테이블로 제공한다.
> 출생일 < 해당 연도 설날이면 → (출생연도 − 1) 기준 띠.

| 동물 | 이모지 | 연도 (mod 12 기준, 1900=쥐) |
|------|--------|----------------------------|
| 쥐 | 🐭 | 0 (1900, 1912 ...) |
| 소 | 🐮 | 1 |
| 호랑이 | 🐯 | 2 |
| 토끼 | 🐰 | 3 |
| 용 | 🐲 | 4 |
| 뱀 | 🐍 | 5 |
| 말 | 🐴 | 6 |
| 양 | 🐑 | 7 |
| 원숭이 | 🐵 | 8 |
| 닭 | 🐔 | 9 |
| 개 | 🐶 | 10 |
| 돼지 | 🐷 | 11 |

### 별자리 카드

- 출생 **월일** 기준 (음력 모드는 변환된 양력 월일 기준) 서양 12궁도 별자리를 표시한다
- 표시 형식: `{기호} {별자리}` (예: `♏ 전갈자리`)

| 별자리 | 기호 | 기간 |
|--------|------|------|
| 염소자리 | ♑ | 12/22 ~ 01/19 |
| 물병자리 | ♒ | 01/20 ~ 02/18 |
| 물고기자리 | ♓ | 02/19 ~ 03/20 |
| 양자리 | ♈ | 03/21 ~ 04/19 |
| 황소자리 | ♉ | 04/20 ~ 05/20 |
| 쌍둥이자리 | ♊ | 05/21 ~ 06/21 |
| 게자리 | ♋ | 06/22 ~ 07/22 |
| 사자자리 | ♌ | 07/23 ~ 08/22 |
| 처녀자리 | ♍ | 08/23 ~ 09/22 |
| 천칭자리 | ♎ | 09/23 ~ 10/23 |
| 전갈자리 | ♏ | 10/24 ~ 11/22 |
| 사수자리 | ♐ | 11/23 ~ 12/21 |

### 태어난 요일 행

- 출생일의 요일을 표시한다 (음력 모드는 변환된 양력 기준)
- 별도 카드 없이 하단에 단일 행 형태로 배치한다

## 예외 / 엣지 케이스

| 상황 | 기대 동작 |
|------|-----------|
| 양력 2월 29일(윤년) 생일 | 비윤년에 다음 생일 계산 시 3월 1일로 대체하여 만 나이·D-day 계산 |
| 연도 변경으로 월의 최대 일수가 줄어드는 경우 | 선택된 일을 해당 월 마지막 날로 자동 클램프 |
| 오늘이 생일 | 다음 생일 카드에 "🎂 오늘이 생일이에요!" 표시, 만 나이는 생일 기준 정상 계산 |
| 오늘 날짜를 생일로 선택 | 만 나이 0세, 살아온 날 0일, 다음 생일 D-day = 내년 생일 |
| 1900년 이전 날짜 선택 시도 | 피커 최솟값(1900년)으로 제한 |
| 음력: 윤달 출생인데 해당 연도 다음 음력 생일에 윤달 없음 | 해당 연도의 평달(윤달 아닌 같은 월)로 대체하여 D-day 계산 |
| 음력: 선택한 연·월 조합에 음력 날짜가 29일까지만 있는데 30일 선택 상태에서 연도 변경 | 29일로 자동 클램프 |
| 음력 변환 룩업 테이블 범위(1900~2100) 밖의 연도 | 1900 또는 2100으로 피커 제한 (실용 범위 이탈 없음) |
| 설날 기준 띠 보정 대상 (양력 1~2월 생) | 해당 연도의 설날 날짜와 출생일을 비교하여 자동 보정, 카드 서브텍스트에 "(설날 기준)" 안내 |

## 비기능 요구사항

- **성능**: 피커 스크롤 시 결과 즉시 업데이트 (1프레임 이내)
- **오프라인**: 네트워크 불필요. 음력 변환 룩업 테이블을 앱에 하드코딩하여 오프라인 완전 동작
- **접근성**: 드럼롤 피커 각 열에 "연도", "월", "일" Semantics 라벨 부여
- **기준일 갱신**: 앱이 백그라운드에서 복귀 시 오늘 날짜를 다시 읽어 결과 재계산
