# Q0015: 실제 단말(아이폰, 안드로이드폰)에 빌드하는 방법 가이드

**날짜**: 2026-03-02
**원본**: [PROPMTS.md](../PROPMTS.md#q0015)

## 질문

1. 자세한 설명이 필요해
2. Simulator에서 아이폰 기종을 변경하는 방법

## 답변

### 1. 실제 단말에 빌드하는 방법

#### iOS (iPhone)

**사전 준비**
- Apple ID 필요 (무료 계정으로도 개발용 빌드 가능, 단 7일마다 재서명 필요)
- Xcode 설치 완료 상태

**단계**

1. **Apple ID 등록** — Xcode > Settings > Accounts > `+` 버튼 > Apple ID 로그인

2. **서명 설정** — Xcode에서 `ios/Runner.xcworkspace` 열기
   - Runner 타겟 > Signing & Capabilities 탭
   - "Automatically manage signing" 체크
   - Team에 본인 Apple ID 선택
   - Bundle Identifier가 고유한지 확인 (예: `com.yourname.calcmate`)

   **xcworkspace 여는 방법:**
   ```bash
   # 명령어로 열기
   open ios/Runner.xcworkspace
   ```
   또는 Finder에서 프로젝트 폴더 > `ios` > `Runner.xcworkspace` 더블클릭.
   `.xcodeproj`가 아니라 반드시 `.xcworkspace`를 열어야 한다 (CocoaPods 의존성 포함).

3. **iPhone 연결** — USB 케이블로 Mac에 연결
   - iPhone에서 "이 컴퓨터를 신뢰하시겠습니까?" > 신뢰

4. **개발자 모드 활성화** (iOS 16+)
   - iPhone > 설정 > 개인정보 보호 및 보안 > 개발자 모드 > 활성화 > 재시작

   **개발자 모드가 안 보이는 경우:**
   - iPhone을 Mac에 USB로 연결한 적이 없으면 메뉴가 숨겨져 있음
   - 해결 순서:
     1. iPhone을 Mac에 USB 케이블로 연결
     2. Xcode 실행 (Xcode가 디바이스를 한 번 인식해야 메뉴가 나타남)
     3. iPhone을 분리한 뒤 재시작
     4. 다시 설정 > 개인정보 보호 및 보안 확인하면 하단에 개발자 모드가 보임

5. **빌드 및 실행**
   ```bash
   flutter devices                    # 연결된 iPhone 확인
   flutter run -d <device-id>         # 해당 디바이스에 실행
   ```
   또는 Android Studio에서 디바이스 선택 후 Run

6. **첫 실행 시 신뢰 설정**
   - iPhone > 설정 > 일반 > VPN 및 기기 관리 > 개발자 앱 > 본인 Apple ID > 신뢰

**무료 vs 유료 Apple Developer 계정 차이**

| 항목 | 무료 | 유료 ($99/년) |
|------|------|---------------|
| 실기기 빌드 | O (7일 유효) | O (1년 유효) |
| App Store 배포 | X | O |
| 등록 가능 기기 수 | 3대 | 100대 |
| Push 알림 | X | O |

#### Android (안드로이드폰)

**사전 준비**
- Android Studio 설치 완료 상태

**단계**

1. **개발자 옵션 활성화**
   - 설정 > 휴대전화 정보 > 소프트웨어 정보 > 빌드번호 7회 탭
   - "개발자 모드가 활성화되었습니다" 표시

2. **USB 디버깅 켜기**
   - 설정 > 개발자 옵션 > USB 디버깅 활성화

3. **USB 연결**
   - USB 케이블로 Mac에 연결
   - "USB 디버깅을 허용하시겠습니까?" > 허용 (항상 허용 체크 권장)

4. **빌드 및 실행**
   ```bash
   flutter devices                    # 연결된 Android 확인
   flutter run -d <device-id>         # 해당 디바이스에 실행
   ```

5. **무선 디버깅** (Android 11+, 케이블 없이)
   - 설정 > 개발자 옵션 > 무선 디버깅 활성화
   - 페어링 코드로 페어링:
   ```bash
   adb pair <ip>:<port>               # 페어링 코드 입력
   adb connect <ip>:<port>            # 연결
   ```

### 2. Simulator에서 아이폰 기종 변경하는 방법

#### 방법 1: 명령어로 관리

```bash
# 사용 가능한 시뮬레이터 목록 확인
xcrun simctl list devices available | grep iPhone

# 특정 기종 부팅
xcrun simctl boot "iPhone 17 Pro Max"

# 시뮬레이터 앱 열기
open -a Simulator

# 시뮬레이터 종료
xcrun simctl shutdown "iPhone 17 Pro Max"

# 전체 종료
xcrun simctl shutdown all
```

#### 방법 2: Simulator 앱에서 변경

1. Simulator 앱 상단 메뉴 > **File** > **Open Simulator** > 원하는 기종 선택
2. 여러 기종을 동시에 열 수 있음

#### 방법 3: Android Studio에서 선택

- 상단 디바이스 드롭다운에서 원하는 시뮬레이터 선택 후 Run

#### 새 시뮬레이터 추가 (목록에 없는 기종)

```bash
# 사용 가능한 기기 타입 확인
xcrun simctl list devicetypes | grep iPhone

# 사용 가능한 런타임 확인
xcrun simctl list runtimes

# 새 시뮬레이터 생성
xcrun simctl create "My iPhone 17" "iPhone 17" "com.apple.CoreSimulator.SimRuntime.iOS-26-2"
```

또는 Xcode > Window > Devices and Simulators > Simulators 탭 > `+` 버튼으로 추가

#### 여러 시뮬레이터에 동시 실행

```bash
# 여러 기종 부팅
xcrun simctl boot "iPhone 16e"
xcrun simctl boot "iPhone 17 Pro Max"

# 모든 디바이스에 앱 실행
flutter run -d all
```

Hot Reload도 모든 디바이스에 동시 적용된다.
