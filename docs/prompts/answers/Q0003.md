# Q0003: Google Stitch에서 디자인된 화면의 디자인된 내용을 적용하는 방법

**날짜**: 2026-02-27
**원본**: [PROPMTS.md](../PROPMTS.md#q0003)

## 질문

1. Figma와 연동하여 export 받아야 하는지
2. Figma 무료 플랜에서도 가능한지
3. 다른 적용 방법이 있는지

## 답변

### Google Stitch 디자인을 Flutter에 적용하는 방법

---

### 1. Figma 연동이 필요한가?

결론부터 말하면 **Figma 연동은 필수가 아니지만, 현재 시점에서 가장 현실적인 경로**다.

**현재 Stitch의 Flutter 지원 상황**

- Stitch는 현재 **HTML/CSS, Tailwind CSS, React** 코드 export를 지원
- **Flutter export는 아직 미지원** (Google이 공식적으로 "향후 추가 예정"이라고 발표)
- 즉, Stitch → Flutter 직접 변환은 지금은 불가능

**현실적인 경로: Stitch → Figma → Flutter**

```
[Google Stitch]
     ↓  "Copy to Figma" (내장 기능)
  [Figma]
     ↓  Figma 플러그인 사용
  [Flutter 코드]
```

---

### 2. Figma 무료 플랜으로 가능한가?

**가능하다.** 단, 아래 제약을 고려해야 한다.

| 항목 | 무료 플랜 |
|------|-----------|
| Stitch → Figma export 수신 | 가능 |
| Figma에서 디자인 편집 | 가능 |
| Figma 플러그인 사용 | 가능 |
| 프로젝트 수 | 3개 제한 |
| 팀 협업 | 제한적 |

CalcMate처럼 혼자 개발하는 경우 **무료 플랜으로 충분하다.**

---

### 3. 전체 적용 방법 3가지

**방법 A: Stitch → Figma → Figma 플러그인 → Flutter**

1. [stitch.withgoogle.com](https://stitch.withgoogle.com)에서 화면 디자인 생성
2. Stitch 우상단 **"Copy to Figma"** 버튼 클릭 (Standard 모드에서만 작동)
3. Figma에서 붙여넣기 후 세부 조정
4. Figma 플러그인 중 하나 사용:
   - **[Figma to Flutter](https://www.figma.com/community/plugin/1236863310201087089)** — AI 기반 Flutter 위젯 코드 생성
   - **[Figma2Flutter](https://www.figma.com/community/plugin/1110606481076006495)** — 자동 UI 위젯 변환
5. 생성된 코드를 프로젝트에 붙여넣고 Riverpod/Clean Architecture 구조에 맞게 정리

**방법 B: Stitch HTML 코드를 참고 자료로만 활용**

- Stitch에서 HTML/CSS export
- 레이아웃·색상·간격 수치를 참고하여 Flutter 위젯을 **직접 작성**
- 가장 번거롭지만 코드 품질 제어가 가장 쉬움
- Clean Architecture를 엄격히 유지해야 할 때 적합

**방법 C: Stitch 화면을 스크린샷 → AI에 넘기기**

- Stitch 미리보기 화면을 캡처
- AI(Claude 또는 Gemini)에 "이 화면을 Flutter 위젯으로 만들어줘"라고 요청
- 빠른 프로토타이핑에 유리

---

### CalcMate 프로젝트에서의 추천 흐름

CalcMate는 디자인 정밀도보다 **로직과 아키텍처**가 핵심인 프로젝트다.

```
Stitch로 화면 레이아웃 아이디어 확인
       ↓
Figma(무료)에 Copy → 색상/여백 수치 확인
       ↓
Flutter 위젯을 직접 작성 (Presentation 레이어)
       ↓
ViewModel과 연결
```

자동 생성 코드는 Clean Architecture 구조와 맞지 않는 경우가 많으므로,
**참고 자료로 활용하고 위젯은 직접 작성**하는 방법 B가 이 프로젝트에 가장 적합하다.
