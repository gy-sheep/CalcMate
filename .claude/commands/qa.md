`docs/prompts/PROPMTS.md` 파일에서 "$ARGUMENTS"에 해당하는 질문을 찾아 답변하는 작업을 수행합니다.

## 단계별 워크플로우

### 1단계: 질문 파악
- `docs/prompts/PROPMTS.md` 파일을 읽습니다
- `$ARGUMENTS`가 ID 형식(Q0001, Q0002...)이면 해당 ID 섹션을 찾고, 아니면 키워드로 질문 본문을 검색합니다
- 여러 개 매칭 시 목록을 보여주고 사용자에게 선택을 요청합니다
- 찾은 질문 전체를 사용자에게 보여주고 "이 질문이 맞나요?" 확인을 요청합니다
- 사용자가 아니라고 하면 중단합니다

### 2단계: 답변 작성
- 사용자가 확인하면 해당 질문에 대한 상세한 답변을 작성합니다
- 코드 예시, 단계별 설명 등을 포함하여 실용적으로 작성합니다

### 3단계: 저장 확인 및 상태 업데이트
- 답변 작성 후 사용자에게 **1회** 저장 여부를 묻습니다: "답변을 저장할까요?"
- 사용자가 **저장한다고 하면**:
  - `docs/prompts/answers/QXXXX.md` 파일을 생성합니다 (QXXXX는 해당 질문 ID)
    - 파일 상단에 질문 원문과 날짜를 포함합니다
    - 그 아래에 답변 전체를 기록합니다
- 사용자가 **저장하지 않는다고 해도**:
  - `docs/prompts/PROPMTS.md`의 해당 섹션 상태는 반드시 업데이트합니다:
  - `**상태**: `미답변`` → `**상태**: `답변완료` | **답변**: [QXXXX.md](answers/QXXXX.md)``로 변경

## answers/QXXXX.md 파일 형식

```markdown
# QXXXX: 질문 제목

**날짜**: YYYY-MM-DD
**원본**: [PROPMTS.md](../PROPMTS.md#qxxxx)

## 질문

질문 원문...

## 답변

답변 내용...
```

## 주의사항
- 해당 ID의 answers/ 파일이 이미 존재하면 덮어쓰기 전에 경고합니다
- 파일에 없는 질문 ID를 입력하면 "해당 ID를 찾을 수 없습니다"라고 안내합니다
